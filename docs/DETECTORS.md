# Detector Packs

V1 detector packs:

- secret-shaped filenames
- destructive git commands
- package-runner bypass
- unsafe deploy/mainnet/production commands
- stale or missing truth files
- missing token-saving tools
- UI/design drift patterns inspired by Impeccable-style deterministic checks

In `observe` mode, detector findings warn. In `guard` and `strict`, high-risk
PreToolUse findings can block.
